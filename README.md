# Topological Kuramoto model on cell complexes

This repository contains the reference implementation for the topological Kuramoto model studied in "Phase locking and multistability in the topological Kuramoto model on cell complexes". It includes routines for constructing cell and simplicial complexes, solving the topological nonlinear Kirchhoff conditions, finding solutions determined by winding numbers, checking their stability, and reproducing the figures in the manuscript.

## Repository structure

- `core/complexes.py`: construction of boundary operators, kernels, and coupling matrices
- `core/dynamics.py`: stationary-state solver, winding-number enumeration, partitions, and stability analysis
- `core/rings.py`: ring-specific routines and figure-generation code
- `core/platonic_solids.py`: generators for Platonic solids as cell complexes
- `core/simplexes.py`: generators for simplicial complexes generated from regular simplexes
- `01_rings.ipynb`, `02_platonic_solids.ipynb`, `03_simplexes.ipynb`, `04_time_series.ipynb`: Jupyter notebooks that reproduce all manuscript results and figures.

## 2. Installation guide

Clone the repository and install the dependencies listed in `requirements.txt`.

### Windows
```bash
git clone https://github.com/ibfzj/topological-kuramoto.git
cd topological-kuramoto
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### macOS/Linux
```bash
git clone https://github.com/ibfzj/topological-kuramoto.git
cd topological-kuramoto
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Typical installation time on a standard desktop or laptop computer is up to 5 minutes, depending on internet connection and Python setup.

## 3. Demo
Demo files are given by the Jupyter notebooks.

Launch Jupyter and run all the cells in the files.

Most cells are evaluated within a few minutes on a standard computer. In the `03_simplexes.ipynb` file, calculations for the complex generated by the 5-simplex take longer (several hours, depending on the computer).

### Reproducing boundary matrices

All boundary matrices and C-matrices for the Platonic solids are found in the file `02_platonic_solids.ipynb`.

All boundary matrices C-matrices for the simplicial complexes generated by regular d-simplexes are found in the file `03_simplexes.ipynb`.


### Reproducing the manuscript figures

The main manuscript figures can be reproduced using the notebooks in the repository:

`01_rings.ipynb`: Figures 3, 4 and 5.

`02_platonic_solids.ipynb`: Figures 6 and 7.

`03_simplexes.ipynb`: Table 1.

`04_time_series.ipynb`: Figure 1 e and f.

Run the notebooks from top to bottom to reproduce the results.
